
How to make angular6 application SEO ready?

Do this once you complete your angular application.

Caution:
Take a backup of the working copy and do the below:

ng add @ng-toolkit/universal
npm run build:prod(If error throws remove all spec.ts and remove jasmine typings)
npm run server (it will run node local.js command. local.js was generated by the above operation)
